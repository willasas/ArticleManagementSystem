## **步骤说明**

**1.按下 WIN+R 键，打开运行窗口，在运行窗口中输入 cmd 回车**
![运行窗口](../img/so_img/run2.jpg)

**2.在命令行窗口中输入对应的 dos 命令即可打开对应的程序/服务**

#### 注意事项

- 有一些命名涉及到权限等问题，需要有管理员身份才能执行！
- 输入的命令不区分大小写
