#### 关键词搜索技巧

**. 任何单词+Tutorial=它的教程**

- eg:Animation tutorial=动画教程
- 3D Animation tutorial=3D 动画教程

- 萌新：在前面的基础上加上 for beginner = 给新手的
- eg：blender tutorial for beginner=blender 的入门教程
