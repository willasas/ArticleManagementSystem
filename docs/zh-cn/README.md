## 文档管理系统

> 一个轻量级的文章管理系统。

## 它是什么

它是一个基于docsify和nodejs的轻量级静态文档管理系统，支持在线评审和全局搜索。

## 特点

- 轻量级
- 支持在线评论
- 全局搜索
- 代码复制
- 离线模式
